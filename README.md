### Команды git
**init** - инициализировать локальный репозиторий  
**add** . - добавить к коммиту  
**commit -m "..."** - закоммитить  
**push origin имя_ветки** - отправить в удаленный  
**pull/fetch origin master** - загрузить в локальный  
**copy** 
**status**- статус коммитов  
**log** - список коммитов  
**branch** - посмотреть активные ветки  
**branch example** - создать ветку example  
**checkout**- переключиться на ветку  
**branch -D имя_ветки** - удалить ветку локально  
**push origin -d имя_ветки** - удалить ветку онлайн
